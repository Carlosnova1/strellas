# 🌟 Strellas

**Strellas** es un juego infantil open source para Android, creado para ser gratis, educativo, divertido y accesible desde cualquier parte del mundo.

## 🎮 Sobre el juego

Strellas es un proyecto de juego infantil pensado para niños y familias.  
La idea principal es crear una experiencia sencilla, colorida y segura, que pueda funcionar en celulares Android sin necesidad de pagar, registrarse o depender de internet todo el tiempo.

El juego estará disponible como archivo **APK**, para que cualquier persona pueda descargarlo e instalarlo en su dispositivo Android.

## 🌍 Objetivo del proyecto

El objetivo de Strellas es crear un juego:

- Gratis
- Open source
- Infantil
- Seguro
- Ligero
- Fácil de instalar
- Compatible con Android
- Disponible globalmente
- Capaz de funcionar sin internet
